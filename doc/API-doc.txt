Регистрация пользователя:

1) http://localhost:8080/users (POST) - регистрация нового пользователя. Формат данных json. Например '{"username":"myuser","password":"mypassword"}'

====================================================================================================================================================

Работа с листами:

1) http://localhost:8080/lists (GET) - получение списка всех туду-листов для текущего пользователя в JSON-формате

2) http://localhost:8080/lists/id (GET) - получение конкретного туду-листа по его идентификатору

3) http://localhost:8080/lists (POST) - публикация туду-листа для текущего пользователя. Формат передаваемых данных - JSON. Например: '{"title": "firstTestList", "elements":[{"text": "Купить яхту", "done":"false"}, {"text": "Завести хомячка", "done":"false"}]}'

4) http://localhost:8080/lists/id (DELETE) - удаление туду-листа по заданному идентификатору

5) http://localhost:8080/lists/element/id/done (PATCH) - пометить элемент листа с данным id как выполненный, если он не выполнен и наоборот
